# Бот который поможет зарабатывать 6к


- Установи nodejs (12.x хотя бы)
- Установи депы - `npm i`
- Скомпиль - `npm run build`
- Запустить можно только с токеном и в POOLING MODE, можешь сгенерить свой или попросить у @xotabu4: `npm start`
